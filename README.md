# Мои заметки по Linux
## Глава 1. Введение

Терминал — это программа, которая запускает оболочку (shell)

Bash — самая распространённая оболочка. Терминал запускает bash внутри себя.

Приглашение ввода ($ - обычный пользователь , # - суперпользователь (root)

## Глава 2. Навигация
Абсолютный путь — от корневого каталога к цели.

Относительный путь — от текущего каталога к цели.

'pwd' — показать рабочий каталог.

'сd'  — сменить каталог. Без аргументов — перейти в домашний.

'cd - ' — переход из рабочего каталога в предыдущий.

'cd ~ ' — переход из рабочего каталог в домашний каталог пользователя.

## Глава 3. Просмотр файлов
ls — список файлов

    ls — просто имена файлов и папок

    ls -l — подробный список (права, владелец, группа, размер, дата)

    ls -a — показать скрытые файлы (начинаются с точки)

    ls -la — подробно + скрытые

    ls -lh — подробно + размеры в читаемом формате (K, M, G)

file — определить тип файла

    file document.pdf  # выведет: PDF document

less — просмотр больших файлов

    less filename — открыть файл

    стрелки / PageUp / PageDown — навигация

    /слово — поиск

    q — выход

## Глава 4. Манипуляции с файлами
cp — копирование

    cp file1.txt file2.txt  # скопировать файл
    cp -r folder1 folder2   # скопировать папку (рекурсивно)

mv — перемещение / переименование
bash

    mv old.txt new.txt  # переименовать
    mv file.txt /home/user/docs/  # переместить

rm — удаление

    rm file.txt   # удалить файл
    rm -r folder  # удалить папку с содержимым
    rm -f file.txt  # удалить без подтверждения (force)

mkdir — создание папок

    mkdir newfolder   # создать папку
    mkdir -p path/to/folder  # создать все папки по пути (если нужно)

Виды ссылок

Жёсткая ссылка (hard link)

    Второе имя для того же файла (один inode)

    Создание: ln target link

    Если удалить исходный файл — данные остаются, доступны через ссылку

    Не работает с папками

Мягкая ссылка (symbolic link, symlink)

    Отдельный файл, который содержит путь к другому файлу или папке

    Создание: ln -s target link

    Если удалить цель — ссылка ломается (broken)

    Может указывать на папку

## Глава 5. Работа с командами

Команда может быть :
    
    Внешняя — файл на диске (ls, cp, grep)
    
    Встроенная — живёт внутри bash (cd, exit, alias, type)
    
    Функцией командной оболочки(shell functions) — это миниатюрные сценарии на языке командной оболочки, встроенные в окружение.
    
    Псевдонимом (alias) — это команда, которую мы можем определить сами, сконструировав из других команд.
    

type — получение типа команды

    type cd       # cd is a shell builtin
    type ls       # ls is aliased или /bin/ls
    type grep     # grep is /usr/bin/grep

which — путь к внешней программе

    which ls      # /bin/ls
    which cd      # (ничего)

help — справка по встроенным командам

    help cd
    help alias

man — руководство

    man ls
    Навигация: стрелки / пробел, поиск: /слово, выход: q

apropos — поиск команд по описанию

    apropos copy    # найдёт cp, scp, rsync...

whatis — краткое описание

    whatis ls       # list directory contents

info — подробная документация

    info ls
        
alias — псевдонимы

    alias ll='ls -la'      # создать
    alias                  # посмотреть все
    unalias ll             # удалить
        
    Временный — существует до закрытия терминала
    Постоянный — прописать в ~/.bashrc
